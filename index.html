How account linking works

1

Parent signs up

A family account is created with a family code.

2

Parent adds children

Each child gets a separate profile and dashboard.

3

Child joins

Child enters family code and links to the correct family.

4

Data stays private

Parents see full details. Siblings and buddies see only streak and progress.

Privacy rule

Every record has a family_id and child_id. A parent only reads data from their own family. A child only sees their own details plus safe summaries for siblings and buddies.
